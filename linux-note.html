1 配置apt软件源

sudo vim /etc/apt/source.list

sudo apt-get update

2 重启ssh服务

sudo service ssh restart

3 查看ssh服务是否启动

sudo lsof -i :22

4 递归改变文件owner

chown -R sun:sun ./dir

5 汇编文件(.s)的结构

.text: 代码段
.data: 常量 字符串 初始化的全局变量
.bss: 不占用实际空间, 仅仅标记那些局部变量和未初始化的全局变量.

因为已知初值的且生命周期长的变量才有提前为它们分配空间的必要,因此data段才占用空间. bss段中记录的是局部变量/未初始化的全局变量, 没必要一开始就为这些量准备好空间, 只要在程序实际运行时为其准备空间即可.

6 makefile

6.1 定义变量

CC := gcc # 定义常量, 后续不能用+=
OBJS = hello.o world.o # 定义变量, 后续可以用+=增加
ObJS += fuck.o

6.2 伪目标 .PHONY

通常是指那些不生成文件的. 只负责执行某些命令的. 

.PHONY: clean  # 这个之后的target都是伪目标

clean: 
        -${RM} *.o ${target}
        echo "finish"

6.3 为整个项目添加宏定义

DEFS = -D_DEBUG -D_POSIX_SOURCE

FLAGS = -g ${DEFS}

6.4 为目标链接上库

LIBS = -lpthread -lqt

6.5 通配符

* 所有
% :任意一个文件
? :

$@ 目标
$^ 全部的依赖文件
$< 首个依赖项 

${target} : ${OBJS}
        ${CC} ${FLAGS} -o $@ $^ ${LIBS}


%.o : %.c
        ${CC} -o $@  $^

6.6 函数调用

$(my_func arg1,arg2,arg3)

6.6.1 当前目录下某一类型的所有文件形成的列表

SRC = $(wildcard *.c)  ==> hello.c world.c fuck.c

6.6.2 将列表中的所有匹配项替换为指定形式

OBJS = $(patsubset %.c,%.o,${SRC} ) 

# 将 SRC = hello.c fuck.c 替换成 hello.o fuck.o
